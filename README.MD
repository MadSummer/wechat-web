# wechat-web
 wechat-web是一个基于nodejs的命令行微信，调用了微信网页版的接口进行封装。
 目前只实现了命令行，之后可以实现网页版或者Electron版本
# 使用方法
1. 安装依赖

    ```js
    npm i
    ```
2. 启动程序

    ```js
    npm run terminal
    ```
